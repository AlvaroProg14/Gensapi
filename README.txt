Bienvenido a Gensapi!

Gensapi es una plataforma web para la creación, análisis y compartición de equipos optimizados mediante un sistema interactivo de arrastrar y soltar, con cálculo automático de sinergias elementales y funcionalidades sociales que permiten a la comunidad descubrir y valorar diferentes composiciones.

Este proyecto ha sido desarrollado en Html, CSS, JavaScript, NODE.js

Para ejecutar el proyecto abre una terminal en la ruta de Gensapi NODE y escribe el comando *node src/index.js*, acto seguido escribe en el navegador "localhost:3000" para desplegar la página

****IMPORTANTE requiere node.js****
